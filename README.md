# Project-Admin-Dashboard
🧠 About the Project

This project is a custom-built admin dashboard designed to practice and demonstrate:

* Advanced CSS Grid layout techniques
* Handling image scaling and containment
* Preventing overflow and layout breaking issues
* Creating a clean and structured UI system

The goal was to build a layout from scratch without relying on frameworks, focusing on understanding how things work under the hood.

🛠️ Built With
* HTML5
* CSS3
* CSS Grid
✨ Features

📐 Fully structured layout using CSS Grid
🖼️ Responsive image handling (object-fit, containment fixes)
📏 Clean spacing and alignment system
🧩 Modular sections:
* Sidebar
* Header
* Main content are
* Announcement / trending sections

🎨 UI details:
* Custom borders and separators
* Typography hierarchy
⚙️ Overflow handling and layout stability improvements
